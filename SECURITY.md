# Security Policy

## Supported Versions

The public release line is `0.1.x`. Security fixes are prepared against the
latest `0.1.x` release candidate until a newer public minor line exists.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub Security Advisories for
`bigmacfive/turbo-graph` when available. If advisories are unavailable, contact
the maintainer privately before opening a public issue.

Include the affected version, platform, a minimal reproduction, and any relevant
crash logs or malformed index files. We will acknowledge valid reports, avoid
public disclosure until a fix is ready, and credit reporters when requested.
